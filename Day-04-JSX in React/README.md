## 📚 Topic

JSX in React

## What is JSX?

JSX stands for JavaScript XML.

It allows us to write HTML-like code inside JavaScript.

## Example

```jsx
function App() {
  return (
    <div>
      <h1>Hello React 🚀</h1>
      <p>This is my first JSX code.</p>
    </div>
  );
}
```

## Run Project

```bash
npm install
npm start
```
