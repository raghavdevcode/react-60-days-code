## 📚 Topic

useState Hook

## What is useState?

useState is a React Hook used to store and update data.

## Example

```jsx
const [count, setCount] = useState(0);
```

```jsx
<button onClick={() => setCount(count + 1)}>
  Increase
</button>
```
